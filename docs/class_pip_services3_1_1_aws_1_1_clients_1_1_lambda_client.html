Detailed Description

Abstract client that calls Lambda function.

When making calls "cmd" parameter determines which what action shall be called, while other parameters are passed to the action itself.

- connections:                   
    - discovery_key:               (optional) a key to retrieve the connection from <see cref="IDiscovery">
    - region:                      (optional) AWS region
- credentials:    
    - store_key:                   (optional) a key to retrieve the credentials from <see cref="ICredentialStore">
    - access_id:                   AWS access/client id
    - access_key:                  AWS access/client id
- options:
    - connect_timeout:             (optional) connection timeout in milliseconds(default: 10 sec)
  • *:logger:*:*:1.0 (optional) components to pass log messages*:counters:*:*:1.0 (optional) components to pass collected measurements
  • *:discovery:*:*:1.0 (optional) services to resolve connection*:credential-store:*:*:1.0 (optional) Credential stores to resolve credentials See , LambdaFunction
